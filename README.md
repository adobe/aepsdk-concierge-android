## About this project

Adobe Experience Platform (AEP) Brand Concierge Extension is an extension for the [Adobe Experience Platform Android SDK](https://github.com/adobe/aepsdk-core-android).

The AEPBrandConcierge extension is used to enable Brand Concierge experiences in your Android app.

For further information about Adobe SDKs, visit the [developer documentation](https://developer.adobe.com/client-sdks/documentation/).

## Requirements

#### Development

##### Open the gradle project file
Open the project in Android Studio by opening the gradle file located at `aepsdk-concierge-android/code`

##### Command line integration

You can run the unit test suites from command line:

~~~
make unit-test
~~~

You can run the android test suites from command line:

~~~
make functional-test
~~~
